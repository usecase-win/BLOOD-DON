About This Project

This Blood Donation Management System is a full-stack mini project built using HTML, CSS, JavaScript (Frontend), Node.js + Express (Backend), and MySQL (Database). It manages 10 database tables covering donors, blood banks, hospitals, blood requests, stock, donation history, volunteers, camps, feedback, and admin users — demonstrating core DBMS concepts like primary keys, foreign keys, joins, constraints, and CRUD operations.
